== Shared transaction header columns ==

DocEntry (int identity), DocNum (int), DocDate, DocDueDate, DocStatus (char 1),
CompanyIDKu, CustmrCode/SuppCode, WhsCode, AuditDate, AuditUser, ObjType

== DocStatus truth table (legacy → new) ==

| Legacy | Meaning in VB6 flows | DocumentStatus | InvoiceStatus |
|--------|-------------------|----------------|---------------|
| O | Open / draft | Draft | Draft |
| C | Closed / posted | Posted | Posted |
| B | Cancelled | Voided | Voided |
| P | Posted (some tables) | Posted | Posted |
| V | Voided (some tables) | Voided | Voided |

== ETL id-map keys ==

- Master: (EntityName, Code) → Guid — e.g. ("Customer", "C001")
- Document: (Division, TableName, DocEntry) → Guid — stored in Entity.LegacyId
- Line: (Division, TableName, DocEntry, LineNum) → Guid
